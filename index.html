About

AI Researcher at INRS Montréal

I work on Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and trustworthy AI systems. My goal is to improve reliability and safety of LLM-based applications using knowledge graphs, advanced retrieval, and multilingual NLP.

My research interests include:

  • Knowledge Graph–enhanced RAG (GraphRAG)
  • Misinformation detection using LLMs
  • LLM robustness, hallucination mitigation, and evaluation
  • Ideological bias analysis in RAG pipelines
  • Multilingual NLP and cross-lingual retrieval

I aim to build production-ready, trustworthy AI systems for real-world applications.

Designed and implemented a knowledge graph–enhanced RAG framework to improve factual accuracy, robustness, and reasoning for LLM applications. Evaluated using the RGB benchmark across noise robustness, counterfactual robustness, and negative rejection.

Built misinformation detection pipelines using classical ML, BERT-based models, and retrieval augmentation. Studied robustness to LLM-generated misinformation and distribution shifts in multilingual settings.
